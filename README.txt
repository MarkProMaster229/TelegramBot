Файл main.py является основным, содержит все команды,
а также некоторые пути к файлам для работы программы.

Команды для работы бота:

/start — запуск.
/pairs — количество пар по группам (7-е задание).
/help — помощь.
/teachers — считает процент выданного ДЗ преподавателем (2-е задание).
/notify — отправка уведомления.
/teacherDZ — проверка на корректность заполнения темы урока (3-е задание).
/analyze_students — проверяет и выдает список студентов с оценкой ниже 3-х (по пятибалльной шкале) (5-е задание).
/assignment_issuance — преподаватели с проверкой ДЗ менее 75%.(1-е задание)
